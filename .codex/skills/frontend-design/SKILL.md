Design Principles:

- Clean and modern
- Mobile first
- Accessible
- Responsive
- Reusable components

Typography:

- Plus Jakarta Sans

Avoid:

- Generic admin templates
- Excessive gradients
- Poor spacing

Prefer:

- ShadCN UI
- Tailwind CSS
- Consistent spacing
- Clear visual hierarchy
